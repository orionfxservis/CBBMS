# Building Management System

A web-based application for managing building operations, tenants, payments, and expenses.

## Features
- **Dashboard**: Overview of income, expenses, and occupancy.
- **Tenant Management**: Add and manage tenant contracts.
- **Payments**: Track rent and maintenance payments.
- **Expenses**: Record and categorize building expenses.
- **Reports**: Generate financial and operational reports.
- **Settings**: Configure building details and preferences.

## How to Run
Simply open `index.html` in your web browser.

## Technologies
- HTML5, CSS3, JavaScript
- Chart.js for visualization
- Google Apps Script (Backend)
